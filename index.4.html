Programação Web

A programação web é justamente esta comunicação, que consegue traduzir as informações para que o computador interprete e devolva um resultado ao usuário final.
Nas palavras do professor e coordenador do curso de Engenharia da Computação da ESEG – Faculdade do Grupo Etapa, Rodrigo Monteiro de Aquino: “A programação web é uma disciplina que ensina conceitos relacionados às tecnologias que permitem a publicação de um website ou sistema computacional web.”
Assim, para aqueles que gostariam de trabalhar como programadores web, é preciso se dedicar a cursos de linguagem de programação ou graduações que contenham disciplinas voltadas para esta finalidade, como é o caso do curso de Engenharia da Computação.
O professor Rodrigo, como coordenador, explica que no curso de Engenharia da Computação da ESEG, existem duas disciplinas que cobram este assunto. “Na primeira, o aluno forma o conhecimento base para a construção de um website simples, porém completo, utilizando as tecnologias de mercado. Na disciplina seguinte é apresentada a arquitetura de um sistema web publicado na internet, e o aluno aprende como desenvolver um website utilizando uma tecnologia moderna de desenvolvimento.”, destaca.
